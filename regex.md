- 一些正则

```tex
/\/$/   //表示以 "/" 结尾
/^$/    //用来匹配空字符串, 如：""
/^[.\/]/ //通过运行得到结果是匹配 . 或者 /
/^\d+(\.\d*)$/  如果是 test(.365) 该运行会先把 .365 转化为字符串 "0.365" 再去执行正则匹配,所以说正则匹配的是需要先转换类型，然后再进行匹配
/^abc|(!)abc/  该正则表达式可以匹配 abc 或者 oo!abc,第一部分匹配 abc 开头，第二部分匹配含有 !abc
/^-?!/  匹配 -! 或者 ！
/^!!/  匹配 !!
/^-!/  匹配 -!
/^-?!+/  匹配 -! 或者 !!! 或者 -！！！！


//特殊的正则表达式
var text = "abc123def", total, left, right;
total = text.replace(/\d+/g, "[$&]"); // abc[123]def
left = text.replace(/\d+/g, "[$`]");  //abc[abc]def
right = text.replace(/\d+/g, "[$']"); // abc[def]def
  
$& - 正则表达式匹配的文本
$` - 匹配文本的左侧内容
$' - 匹配文本的右侧内容
```
