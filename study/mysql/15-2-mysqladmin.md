在 MySQL 8 中，您可以使用 `mysqladmin` 命令来查看各种配置信息和服务器状态。以下是一些常用的 `mysqladmin` 命令及其简短描述：

1. **查看服务器状态**：
   ```sh
   mysqladmin status
   ```

2. **显示服务器变量**：
   ```sh
   mysqladmin variables
   ```

3. **显示服务器统计信息**：
   ```sh
   mysqladmin extended-status
   ```

4. **查看进程列表**：
   ```sh
   mysqladmin processlist
   ```

5. **显示服务器配置选项**：
   ```sh
   mysqladmin variables
   ```

6. **检查服务器是否运行**：
   ```sh
   mysqladmin ping
   ```

7. **查看服务器版本**：
   ```sh
   mysqladmin version
   ```

这些命令可以帮助您快速获取 MySQL 服务器的配置信息和状态。
