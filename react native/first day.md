- 网页运行的示例

```tex
1、能在网页运行是因为“沙盒环境(Snack Play)”，它是由 Expo 公司提供的专门用于演示和学习 React Native 的交互工具；
2、import { Text, View } from 'react-native'; // Text 用来放置文本；View 相当于 HTML 的 Div;
3、至于样式：<View style={{ flex: 1, justifyContent: "center", alignItems: "center" }}></View>；// 使用 style 属性，将 json 数据放置其中， {{ }} 包裹着，之所以有两层，原因是因为外面一层里面的是一个对象；
4、ReactNative中，变量的命名字面意义是和 web 的接近的，Android 和 IOS 中则另有一套它们的规则；
```
